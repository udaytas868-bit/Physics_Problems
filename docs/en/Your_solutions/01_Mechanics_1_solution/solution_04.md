# Problem 4: Vector Calculus

## Formulas
Velocity: $\vec{v}(t) = \frac{d\vec{r}}{dt}$
Acceleration: $\vec{a}(t) = \frac{d\vec{v}}{dt}$

---

## Solution
Given: $\vec{r}(t) = (3t^2)\hat{i} + (5t - 8t^2)\hat{j}$

1. **Velocity vector:**
$$\vec{v}(t) = \frac{d}{dt}(3t^2)\hat{i} + \frac{d}{dt}(5t - 8t^2)\hat{j} = (6t)\hat{i} + (5 - 16t)\hat{j}$$

2. **Acceleration vector:**
$$\vec{a}(t) = \frac{d}{dt}(6t)\hat{i} + \frac{d}{dt}(5 - 16t)\hat{j} = (6)\hat{i} + (-16)\hat{j}$$
