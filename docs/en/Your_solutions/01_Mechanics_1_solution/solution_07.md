# Problem 7: Elimination of Time

## Solution
Given: $x(t) = 2t^2, \quad y(t) = 3t^3$

1. **Eliminate $t$:**
$t^2 = \frac{x}{2} \implies t = \left(\frac{x}{2}\right)^{1/2}$
$y = 3\left(\frac{x}{2}\right)^{3/2}$

2. **Vectors:**
$\vec{v}(t) = (4t, 9t^2), \quad |\vec{v}(t)| = \sqrt{16t^2 + 81t^4}$
$\vec{a}(t) = (4, 18t), \quad |\vec{a}(t)| = \sqrt{16 + 324t^2}$

3. **Interpretation:**
The acceleration is **not** constant because $\vec{a}(t)$ depends on time $t$.
